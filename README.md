# 🎬 Netflix Recommendation System & Analytics Dashboard

## 📌 Project Overview

This project is a Netflix-style recommendation and analytics system that combines Machine Learning with Business Intelligence.

The project builds a movie recommendation engine using Python and creates an interactive Power BI dashboard to analyze Netflix content trends.

The system recommends movies based on user preferences and movie similarities while the dashboard provides insights about Netflix movies, TV shows, genres, ratings, countries, and release trends.

---

## 🚀 Features

### 🤖 Machine Learning Recommendation System

- Content-Based Recommendation
- Collaborative Filtering
- Hybrid Recommendation System
- User-Movie Rating Matrix
- Movie Similarity Analysis
- Personalized Movie Suggestions


### 📊 Power BI Dashboard

- Total Netflix Movies & TV Shows Analysis
- Genre Distribution
- Rating Analysis
- Top Countries by Content
- Movies vs TV Shows Comparison
- Release Year Trends
- Interactive Visualizations


---

## 🧠 Machine Learning Techniques Used

### Content-Based Filtering

Recommends movies by comparing movie features.

Techniques:

- TF-IDF Vectorization
- Cosine Similarity


### Collaborative Filtering

Finds users with similar interests.

Techniques:

- User-Movie Matrix
- Matrix Factorization
- SVD Algorithm


### Hybrid Recommendation

Combines content-based and collaborative filtering to improve recommendations.
